# Guardrail HR Documentation Index

This folder contains product, UX, engineering, pricing, brand, legal-safety, and archive documentation for Guardrail HR. Docs are organized by purpose so contributors can find the right source before editing app logic or product-sensitive areas.

---

## How to Use These Docs

- Start with **product docs** for intent and feature definitions.
- Use **UX docs** for interface decisions and flow rationale.
- Use **engineering docs** for implementation details and system behavior.
- Use **pricing** and **legal-safety docs** before touching gating, disclaimers, or compliance-adjacent copy.
- Use **archive docs** only for historical context — these are superseded and not authoritative.
